# MCB Website

Static landing page for MCB - showcasing the platform and providing desktop app downloads.

## Development

```bash
# Install dependencies
npm install

# Start development server
npm start

# Build for production
npm run build
```

## Deployment

### GitHub Pages
```bash
# Update homepage in package.json
# Then deploy
npm run deploy
```

### Netlify
```bash
npm run build
# Upload the build folder to Netlify
```

## Features

- Responsive landing page
- Download links for desktop app (macOS, Windows, Linux)
- Mission and solutions showcase
- Blog section
- Contact information

## Tech Stack

- React 19
- TypeScript
- Tailwind CSS
- Framer Motion (animations)
- React Router
