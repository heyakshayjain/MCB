# MCB Flask App

This converts your existing static frontend into a minimal Flask app.

Quick start (macOS, zsh):

1. Create a virtual environment:

```bash
python3 -m venv venv
```

2. Activate it (zsh):

```bash
source venv/bin/activate
```

3. Install dependencies:

```bash
pip install -r requirements.txt
```

4. Run the app:

```bash
python app.py
```

Open http://127.0.0.1:5000/
